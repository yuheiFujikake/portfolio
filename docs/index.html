Profile

専門学生時代に学費を自分で支払うためアルバイトと学業を両立をしながら国家資格であるITパスポート・基本情報技術者試験に合格し2年間の学業を経て2016年に専門学校を卒業する。

専門学校卒業後、独立系SIer企業へ就職し約2年間の金融機関による保険商品販売支援Webアプリケーションシステム開発に携わり、その後2019年1月に退社するまでPoCとして鉄道会社による切符販売支援Android,iOSアプリ開発に携わる。

翌月にWeb制作会社へ転職し、大手クライアントのコーポレートサイトのフルリニューアルやランディングページの制作に携わり、最近では過去のAndroidアプリの業務経験により自社製品システムの開発に携わる。
